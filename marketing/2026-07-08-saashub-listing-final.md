## Description (EN, ~130 words)

```
BFSG-Fuchs is an automated accessibility scanner for German websites 
(BFSG / WCAG 2.1 AA / EAA). It renders each page in a real browser 
(Playwright) and runs axe-core checks against WCAG 2.1 AA — the technical 
baseline referenced by Germany's Barrierefreiheitsstärkungsgesetz (BFSG) 
via EN 301 549, and by the EU's European Accessibility Act (EAA). A free instant check returns a score and 
finding count in about 60 seconds, no signup required. Paid reports (from 
€129 one-time) add a prioritized PDF with source-level findings and a draft 
accessibility statement; the Profi tier covers up to 25 subpages plus a 
remediation plan with developer notes. A separate check reviews cookie/ 
consent banners under Germany's TDDDG. No overlay widget — findings point to 
the actual markup so teams can fix it. Every paid report is reviewed by the 
founder before delivery. Hosted in Germany (Hetzner, Nuremberg).
```

```
1. axe Monitor (Deque) — automated + manual accessibility monitoring, 
   broader enterprise scope
2. Siteimprove — enterprise-wide digital governance suite incl. 
   accessibility
3. accessiBe — overlay-widget approach; BFSG-Fuchs instead audits the 
   rendered source and does not inject a widget
4. WAVE (WebAIM) — free browser-based single-page checker, no PDF report or 
   German-market focus
```

**Hinweis Formulierung Punkt 3:** "instead audits… and does not inject a widget" ist eine
objektiv-technische Unterscheidung (Overlay vs. Quellcode-Audit), keine Abwertung des
Wettbewerbers — UWG-§-6-konform (vergleichende Werbung nur mit objektiv messbaren, nicht
herabsetzenden Aussagen).
