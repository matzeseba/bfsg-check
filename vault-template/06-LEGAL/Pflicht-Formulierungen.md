# Pflicht-Formulierungen & No-Gos

## Immer verwenden (Whitelist)

| Kontext | Korrekte Formulierung |
|---|---|
| Produkt-Beschreibung | „automatisierte technische Analyse" |
| Audit-Art | „WCAG-2.1-AA-Audit" |
| Ergebnis-Beschreibung | „identifiziert technische Barrieren gemäß WCAG 2.1 AA" |
| Scan-Resultat | „technischer Prüfbericht" |
| Disclaimer-Footer | Exakter Wortlaut: `docs/legal-templates/disclaimer-footer.md` |

## Niemals verwenden (Blacklist — automatisch geblockt)

| Begriff | Warum verboten |
|---|---|
| „BFSG-konform" | UWG §5 Irreführung — wir können keine Konformität garantieren |
| „rechtssicher" | UWG §5 Irreführung |
| „garantiert" | UWG §5 Irreführung |
| „TÜV-geprüft" / „DEKRA" | Ohne Zertifizierung unlautere Werbung |
| „100% barrierefrei" | Unmöglich zu garantieren |

## Disclaimer-Footer (Kurzform)

> „Dieser Bericht ist eine automatisierte technische Analyse auf Basis von WCAG 2.1 AA und stellt keine Rechtsberatung dar. Eine vollständige Barrierefreiheit kann durch automatisierte Tests allein nicht garantiert werden."

Vollständiger Wortlaut: `docs/legal-templates/disclaimer-footer.md` im Repo.

## Vergleichende Werbung (erlaubt, wenn…)

§ 6 UWG erlaubt Vergleiche, wenn:
1. Objektiv messbare Kriterien (WCAG-Score, gefundene Fehler-Anzahl)
2. Nicht herabsetzend
3. Kein Wettbewerber namentlich genannt ohne Zustimmung
